## 核心思想

如果叫你来创造一个像今天一样的宇宙，你会怎么做？

方案一，用巨大无比的计算机去每时每刻写入这些变量和参数，去设计原子、粒子、天体、生命…，结果发现，这些变量和参数的量会多到无穷大个宇宙都装不下。

方案二，就是让——最小单元方向守恒，然后，让这些最小单元相遇只会耦合，不会折射和反射；剩下的一切活动，让它自己去宇宙里跑，能跑通，路径能闭合变成稳定态，它就活；跑不通，路径不能闭合，不能变成稳定态，它就死。

显然，任何宇宙制造者都会选择方案二；宇宙也不笨，于是它选择了方案二。

宇宙第一性公理（方向守恒公理）

宇宙中，存在最小单元，具有初始运动方向并方向守恒；两个以上最小单元耦合，构成结构，其结构运动方向是其合成方向；当其解耦， 各最小单元回归各自的初始运动方向。

###############

宇宙并非一个空白容器。宇宙是允许，最小单元的守恒的方向路径——经过的空间。这些最小单元的守恒的方向路径，事实上，就是一种几何格式，因为它们都是直线（不管它们初始于哪一个方向），但在宇宙中，这些直线就是事实上的几何。然后，最小单元耦合成为结构后，它们会合成方向，但这些合成方向在宇宙几何格式坐标中，就表现为：或斜线或弧线或曲线或椭圆...，这就是我们接下来要建立的——格式化几何宇宙的模型。

## 格式化几何宇宙的模型最底层是直线

因为宇宙第一性公理规定了，最小单元方向恒定、不允许折射和反射<br>
在这种情况下：<br>
方向 = 不变的指向<br>
路径 = 指向在宇宙中的展开<br>
所以，宇宙底层几何首先是一张直线网络，因为：<br>
•	每一条最小方向路径都会留下不可抹除的记录<br>
•	这些记录共同构成宇宙的“几何底图”

宇宙的最底层几何不是欧几里得假设，而是方向路径的历史总和。

在我们的格式化几何宇宙的模型中，任何以曲线（轨道）运行的结构，一定是“合成方向”的表现。因为：

•	当两个或多个最小单元耦合成结构<br>
•	各自的直线方向不能被消除<br>
•	只能被同时保留

于是，在宇宙几何坐标中：

•	局部看：方向在变化<br>
•	整体看：每一个最小方向都在守恒

这在几何上只能表现为：

•	弧线<br>
•	曲线<br>
•	闭合曲线<br>
•	椭圆、螺旋……

这些曲率是结构“多方向守恒的必然结局”。
