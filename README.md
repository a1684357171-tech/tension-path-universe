## 几何格式化宇宙核心思想

如果叫你来创造一个像今天一样的宇宙，你会怎么做？

方案一，用巨大无比的计算机去每时每刻写入这些变量和参数，去设计原子、粒子、天体、生命…，结果发现，这些变量和参数的量会多到无穷大个宇宙都装不下。

方案二，就是让宇宙中的——最小单元运动方向守恒；然后，让这些最小单元方向相遇只会耦合，不允许折射和反射；再然后，当耦合到宇宙空间没有可选几何路径(饱和态后)并继续耦合时，原有耦合态必然解耦（辐射或爆炸）。接下来，剩下的一切活动，就让它自己去宇宙里跑，能跑通，路径能闭合变成稳定态，它就活（形成结构态）；跑不通，路径不能闭合，不能变成结构态，它就死。

显然，任何宇宙制造者都会选择方案二；宇宙也不笨，它选择了方案二。

## 为什么要规定这些最小单元初始方向守恒，相遇只会耦合，不允许折射和反射

如果这些最小单元初始方向不守恒，它们相遇耦合后就不会出现稳定的方向合成或轨道，以及，当它们的结构解耦后就会出现大家都随机乱跑的致命错误，这个世界就没有稳定的东西了。这跟我们现在的世界天体运行轨道稳定，生命结构稳定，元素周期稳定等事实不符。所以，宇宙最小单元，它们的初始方向必须守恒和不允许乱跑。

如果允许它们相遇可以折射和反射，那么显然，它们就永远不会耦合成结构，这个宇宙也不会形成任何结构，我们今天就不会在这里。

## 为什么要规定当结构耦合到宇宙空间没有可选几何路径(饱和态)并继续耦合时，原有耦合态必然解耦（辐射或爆炸）。

这是因为，如果宇宙自己不生成这样的规则，宇宙会被这些不断耦合的结构挤压死，我们没有今天 。

## 这些规定不是我们人为给它假设的，而是宇宙如果要有东西存在的话，在逻辑上，它必然要这样做。

所以，我们就可以给它确定一个无需证明就自然成立的公理化形式：

## 宇宙第一性公理（一）（方向守恒公理）

宇宙中，存在最小单元，具有初始运动方向并方向守恒；两个以上最小单元耦合，构成结构，其结构运动方向是其合成方向；当其解耦， 各最小单元回归各自的初始运动方向。

## 宇宙第一性公理（二）（耦合公理）

宇宙中两个以上最小单元方向相遇，只允许耦合，不允许反射和折射。

## 宇宙第一性公理（三）（解耦公理）

当结构耦合到宇宙空间没有可选几何路径(路径饱和)并继续耦合时，原有耦合态解耦（辐射或爆炸）

--------
从“宇宙第一性公理（一）”我们还可以自然推出，设第一个最小单元为 a ,第二个最小单元为 b, 先假设它们是同方向平行运动的，因为它们方向守恒，显然，a 和 b 之间的间隔距离s，在宇宙里一定守恒。那么，这个 s 就是宇宙几何网格密度；显然，如果一个或几个第一性结构体，在这个 s 空间里，就不会再跟其他（比如c）最小单元“碰撞”，所以，这个宇宙网格空间 s 是结构态的——“安全存在空间”。

所以，我们还需要一个：

## 宇宙第一性公理（四）（网格化公理）

宇宙空间是以网格化方式存在的，这些网格的边界是宇宙最小单元路径，这些网格之间的几何距离是s。


## 宇宙是什么

宇宙并非一个空白容器。宇宙是允许，最小单元的守恒的方向路径——经过的空间。这些最小单元的守恒的方向路径，事实上，就是一种几何格式，因为它们都是直线（不管它们初始于哪一个方向），但在宇宙中，这些直线就是事实上的几何。然后，最小单元耦合成为结构后，它们会合成方向，但这些合成方向在宇宙几何格式坐标中，就表现为：或斜线或弧线或曲线或椭圆...这就是我们接下来要建立的——几何格式化宇宙的模型；通过这个模型，我们就可以直接一步步用几何语言推演出今天的一切，而不再需要其他假设语言和假设性补丁。
